## Getting Started

Clone the repository

```bash
git clone https://github.com/Harshalkatakiya/invoice-maker.git
cd invoice-maker
```

### Install dependencies

Open two terminals for frontend and backend.

in 1st terminal:

```bash
cd frontend
npm install
npm run dev
```

in 2nd terminal:

```bash
cd backend
npm install
npm run dev
```
